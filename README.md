

This project is a real-time object detection web application built using YOLOv8 and Streamlit.

## 🚀 Features

* Webcam-based object detection
* Fast and accurate detection using YOLOv8
* Simple and clean UI
* Download detected images

## 🛠️ Tech Stack

* Python
* YOLOv8 (Ultralytics)
* Streamlit
* OpenCV

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📸 Use Case

* Smart surveillance systems
* AI-based monitoring
* Computer vision projects
